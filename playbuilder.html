Getting Started

Getting Started

  1. Download and install the Playbuilder Chrome extension
  2. Login to the AudienceStream UI
  3. Open up the newly installed extension

Creating a Simple Play

This play generates one badge, and contains one question that asks the users of this play to define the qualifications of a VIP visitor.

Step One, Create a Badge Attribute

  1. Create a new attribute
  2. Configure the badge template

Step Two, Assign an Enrichment to the Badge

  1. Create a new enrichment
  2. Configure the assign badge enrichment

Step Three, Create a Rule Placeholder

  1. Create a new rule
  2. Configure the rule. This rule will be used as a placeholder for the custom rule that the user will specify

Step Four, Putting It All Together and Creating the Play

  1. Go to the Plays tab
  2. Add a play
  3. Configure the play
  4. Create one question for the play
  5. Accept the changes and save the play

Final Step, Save and Publish Your Changes

  1. Click on "Save & Publish"
  2. Complete the publish
